üìñ Definition: Classification means arranging elements in groups based on their similar properties. This makes studying 118 elements easier and more systematic.

2Ô∏è‚É£ Dobereiner's Triads (1817)

üìñ Definition: Triads are groups of THREE elements with similar properties. The atomic mass of middle element = Average of first and third element.
üí° Simple Concept
  • What? Take 3 similar elements (like Li, Na, K - all soft metals)
  • Magic Rule: Middle one's mass = (First + Last) √∑ 2
  • Example: Li(6.9), Na(23), K(39.1) ‚Üí Check: (6.9+39.1)√∑2 = 23 ‚úì
Middle Element = (Element 1 + Element 3) √∑ 2

üìù Example:

3Ô∏è‚É£ Newlands' Law of Octaves (1866)

üìñ Definition: When elements are arranged by increasing atomic mass, every 8th element has similar properties - like musical notes Sa Re Ga Ma (repeating after 8).
üí° Musical Connection
  • Like Music: Sa Re Ga Ma Pa Dha Ni Sa (8 notes repeat!)
  • Elements: 1st and 8th similar, 2nd and 9th similar... and so on
  • Example: H and F both reactive, Li and Na both soft metals
üéµ Musical Pattern:

                ‚Üì   ‚Üì   ‚Üì   ‚Üì   ‚Üì   ‚Üì    ‚Üì   ‚Üì
Elements:      1st 2nd 3rd 4th 5th 6th  7th (8th same as 1st!)
üé™ Memory Trick: "Every 8th element repeats!"

üìä Newlands' Arrangement:

Row 1 H F Cl Co,Ni Br ‚Äî ‚Äî

‚úÖ Merits
  • Creative idea
  • Easy to remember
  • Worked for light elements
‚ùå Demerits
  • Only till Calcium (Ca)
  • No space for new elements
  • Two elements in one box!
üìù Board Question (2 Marks):
Q. Why was Newlands' Law of Octaves rejected?
Answer:
1. Valid only up to Calcium
2. No provision for newly discovered elements
3. Placed dissimilar elements together (like Co & Ni with halogens)

4Ô∏è‚É£ Mendeleev's Periodic Table (1869)

üìñ Mendeleev's Periodic Law: "Properties of elements are periodic function of their ATOMIC MASSES"
(Elements with similar mass show similar properties)
üí° Mendeleev's Smart Thinking
  • Arranged: 63 elements by increasing atomic mass
  • Structure: 7 Periods (rows) and 8 Groups (columns)
  • Smart Move: Left GAPS for undiscovered elements!
  • Prediction: Even told properties of unknown elements!
üèÜ Who was Mendeleev? Russian scientist (1834-1907) - Called "Father of Periodic Table"

üåü Amazing Predictions:

Mendeleev Called It Year Found Real Name Accuracy
Eka-boron (44) 1879 Scandium (Sc) ‚úì Perfect!
Eka-aluminium (68) 1875 Gallium (Ga) ‚úì Perfect!
Eka-silicon (72) 1886 Germanium (Ge) ‚úì Perfect!
üé™ Memory Trick: "Sir Gave Great" predictions
Scandium, Gallium, Germanium!
‚úÖ Merits
  • Left Gaps: For unknown elements (smart!)
  • Predictions: Told properties before discovery
  • Corrections: Fixed wrong atomic masses
  • Noble Gases: Added later without problem
‚ùå Demerits
  • Hydrogen: Where to place? Group 1 or 7?
  • Isotopes: Cl-35 and Cl-37 where to place?
  • Co-Ni Problem: Co(58.9) before Ni(58.7)?
  • No Pattern: Can't predict gaps between elements
üìù Board Question (4 Marks):
Q. State two merits and two demerits of Mendeleev's periodic table.
Answer:
Merits:
1. Left gaps for undiscovered elements and predicted their properties accurately
2. Corrected atomic masses of some elements (Be: 14‚Üí9.4)

Demerits:
1. Position of hydrogen was uncertain (Group 1 or 7?)
2. Couldn't place isotopes properly (same properties, different mass)

5Ô∏è‚É£ Modern Periodic Table (1913)

üìñ Modern Periodic Law: "Properties of elements are periodic function of their ATOMIC NUMBERS"
(Not mass - but number of protons!)
üí° The Big Change
  • Who? Henry Moseley (British scientist) using X-rays
  • What Changed? Atomic NUMBER instead of atomic MASS
  • Why Better? Atomic number is unique for each element!
  • Result: Solved all Mendeleev's problems ‚úì
Atomic Number = Number of Protons = Unique for every element!
Example: All Chlorine atoms (Cl-35, Cl-37) have atomic number 17 ‚Üí Same place!

üèóÔ∏è Structure:

üìù Board Question (3 Marks):
Q. State Modern Periodic Law. How does it remove Mendeleev's problems?
Law: Properties are periodic function of atomic numbers.

Problems Removed:

2. Co-Ni: Correct order by atomic number (27, 28)

6Ô∏è‚É£ Electronic Configuration & Periodic Table

üìñ Connection: Electronic configuration (electron arrangement) decides element's position in periodic table.
üí° Golden Rules
  • Group Number = Valence Electrons (electrons in outer shell)
  • Period Number = Number of Shells (how many shells)
  • Example: Na (2,8,1) ‚Üí 3 shells = Period 3, 1 valence = Group 1
Period = Number of Shells
Group = Valence Electrons

üìä Examples:

Element At. No. Config Shells Valence Period Group

üìù Practice Example:

Q: Element X has config 2,8,6. Find Period, Group, Valency?

Solution:
‚Ä¢ Shells = 3 ‚Üí Period = 3 ‚úì
‚Ä¢ Valence electrons = 6 ‚Üí Group = 16 ‚úì
‚Ä¢ Valency = 8-6 = 2 ‚úì

üìù Board Question (3 Marks):
Q. Electronic configuration of element is 2,8,5. Determine: (a) Period (b) Group (c) Valency
Answer:
(a) Period = 3 (3 shells: K, L, M)
(b) Group = 15 (5 valence electrons)
(c) Valency = 3 (8-5=3 or 5, both possible)

7Ô∏è‚É£ Periodic Trends - Very Important!

üìñ Periodic Trends: Patterns in properties that repeat in the periodic table. Many board questions come from this!

üìà Trend 1: VALENCY

üí° What is Valency?
  • Definition: Combining capacity (how many bonds it can form)
  • Depends on: Valence electrons (electrons in outer shell)
  • Pattern: 1‚Üí2‚Üí3‚Üí4‚Üí3‚Üí2‚Üí1‚Üí0 (Groups 1 to 18)
Across Period: 1-2-3-4-3-2-1-0
Down Group: SAME valency
üé™ Remember: "1234321 Zero"

üìà Trend 2: ATOMIC SIZE

üí° How Does Size Change?
  • Across Period (‚Üí): Size DECREASES (gets smaller)
  • Why? Nuclear charge increases, pulls electrons closer
  • Down Group (‚Üì): Size INCREASES (gets bigger)
  • Why? New shell added, distance increases
Right ‚Üí Size decreases ‚Üì
Down ‚Üì Size increases ‚Üë
üìù Examples:

Period 3: Na > Mg > Al > Si > P > S > Cl (size decreases ‚Üí)

Group 1: Li < Na < K < Rb < Cs (size increases ‚Üì)

üìù Board Question (2 Marks):
Q. Arrange in increasing atomic radius: O, C, F, N
Answer: F < O < N < C
(All in Period 2, Right to Left size increases)

üìà Trend 3: METALLIC CHARACTER

üí° Metal vs Nonmetal
  • Metallic: Ability to lose electrons easily
  • Nonmetallic: Ability to gain electrons easily
  • Across Period (‚Üí): Metal changes to Nonmetal
  • Down Group (‚Üì): Becomes more Metallic
Right ‚Üí Nonmetallic increases
Down ‚Üì Metallic increases
üé™ Remember:
"Metals love Bottom-Left corner"
"Nonmetals love Top-Right corner"
üìù Board Question (3 Marks):
Q. Arrange in increasing metallic character: Mg, Na, Al, Si
Answer: Si < Al < Mg < Na

Reason: All in Period 3, Right to Left metallic character increases
